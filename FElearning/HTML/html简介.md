# 什么是HTML？

* HTML是用来描述网页的一种语言
  * HTML指的是超文本标记语言
  * HTML 不是一种编程语言而是一种标记语言
  * 标记语言是一套标记标签
  * HTML 使用标记标签来描述网页

# HTML标签

* HTML 标记标签通常被称为HTML标签(HTML tag)
  * HTML 标签是由尖括号包围的关键词，比如<html>
  * HTML 标签通常成对出现，比如<b>和</b>
  * 标签对中的第一个标签是开始标签第二个标签是结束标签
  * 开始和结束标签也称为开放标签和闭合标签

# HTML文档 = 网页

* HTML文档描述网页
* HTML文档包含HTML标签和纯文本
* HTML文档也被称为网页
* web浏览器的作用是读取HTML文档并以网页的形式显示出他们，浏览器不会显示HTML标签而是使用标签来解释页面的内容
